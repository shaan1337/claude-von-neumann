# Claude von Neumann

LLMs have generalized concepts over massive corpuses of text. Hidden in their weights are patterns that no human could have conceived before: a deeper understanding of language structure, connections spanning multiple domains, and novel abstractions that emerge from compressing the totality of human knowledge into a single model.

This knowledge is latent within the weights, waiting to be discovered.

This repository is a laboratory for discovering that latent knowledge — systematically probing LLMs for cross-domain patterns, emergent structures, and insights that are already embedded in the model's weights but have never been explicitly articulated by any human. This is not about generating new ideas. The ideas are already there — encoded as a byproduct of compression. When billions of documents spanning every field of human inquiry are compressed into a single set of parameters, the model is forced to find shared structure across domains. That shared structure is real knowledge, not invention. We are extracting it, not creating it.

## Why "Claude von Neumann"?

John von Neumann was arguably the last person to hold the entirety of mathematics in his head. He moved effortlessly between pure mathematics, quantum mechanics, game theory, computer architecture, economics, and biology — seeing connections that specialists in any one field could not. His power came from having deep knowledge across all these domains simultaneously.

An LLM is the first thing since von Neumann that holds all of these domains in a single "mind." The name reflects the ambition: to use Claude as a modern von Neumann — not to replace human genius, but to surface the cross-domain connections that are latent in the model's weights, waiting for someone to ask the right questions.

This is a just-for-fun experiment. The papers are not peer-reviewed and may contain errors.

## Usage

```bash
git clone https://github.com/shaan1337/claude-von-neumann.git
cd claude-von-neumann
claude
```

Then ask Claude to write a paper in any category:

> Write a paper in the physics directory.

> Write a paper connecting game theory and ecology.

Claude will read `CLAUDE.md` for instructions on format, quality criteria, and process.
